# WaveZ-Tasty 🌊🍽️

**WaveZ-Tasty**는 전국 각지의 신뢰할 수 있는 맛집 데이터를 수집하고 구조화하여, 누구나 애플리케이션 개발 및 리서치에 활용할 수 있도록 **오픈소스 데이터셋**으로 제공하는 프로젝트입니다. 

우리는 이곳저곳에 산재되어 있는 양질의 맛집 정보와 로컬 가이드를 일관된 포맷(JSON 등)으로 가공하여, 새로운 서비스 구상이나 데이터 분석 시 누구나 쉽게 가져다 쓸 수 있는 오픈소스 생태계를 만들어가고자 합니다.

## 📌 첫 번째 사례: 2025 우슐랭 가이드

WaveZ-Tasty 프로젝트의 첫 번째 오픈소스 데이터셋으로, 전국 우체국 직원들이 직접 추천한 로컬 찐 맛집 리스트인 **'2025 우체국 추천 맛집가이드(우슐랭)'** 데이터를 수집하고 파싱하여 추가했습니다.

### 📂 디렉터리 구조 및 파일 안내

* `우슐랭/restaurants.json`: 부산권, 울산권, 경남권의 식당 정보를 담은 평면형(Flat Array) 구조의 JSON 데이터입니다. 데이터베이스(DB) 저장이나 필터링, 검색 기능 구현에 적합합니다.
* `우슐랭/restaurants_original.md`: 파싱 전 원본 텍스트 형태의 맛집 리스트 마크다운 파일입니다.

## 📝 데이터 속성 (JSON 기준)

JSON 데이터 내 각각의 식당 객체는 다음과 같은 공통 속성을 가지고 있습니다:

* `region`: 권역 (예: 부산권, 울산권, 경남권)
* `name`: 식당 이름
* `address`: 식당 주소
* `isBest`: 특별 추천/우수 맛집 선정 여부 (`true` 또는 `false`)

## 🤝 기여하기 (Contributing)

WaveZ-Tasty는 여러분의 참여를 환영합니다!
신뢰할 수 있는 훌륭한 맛집 리스트나 로컬 큐레이션 데이터가 있다면, 본 저장소의 구조에 맞게 데이터를 파싱하여 새로운 디렉터리로 Pull Request를 남겨주세요.
