# WaveZ-Tasty 🌊🍽️

이 저장소는 **'2025 우체국 추천 맛집가이드'** (일명 우슐랭) 데이터를 WaveZ 프로젝트에서 쉽게 활용할 수 있도록 파싱 및 구조화하여 모아둔 데이터 저장소입니다.

## 📂 디렉터리 구조 및 파일 안내

* `우슐랭/restaurants.json`: 부산권, 울산권, 경남권의 식당 정보를 담은 평면형(Flat Array) 구조의 JSON 데이터입니다. 데이터베이스(DB) 저장이나 필터링, 검색 구현에 적합합니다.
* `우슐랭/restaurants_original.md`: 원본 텍스트 형태의 맛집 리스트 마크다운 파일입니다.

## 📝 데이터 속성 (JSON 기준)

JSON 파일 내 각각의 식당 데이터는 다음과 같은 속성을 가지고 있습니다:

* `region`: 권역 (부산권, 울산권, 경남권)
* `name`: 식당 이름
* `address`: 식당 주소
* `isBest`: BEST (우수 맛집) 선정 여부 (`true` 또는 `false`)

## 🚀 활용 방법

본 저장소의 JSON 파일을 불러와서 프론트엔드의 화면에 리스트로 렌더링하거나, 백엔드의 데이터베이스 초기화(Seeding) 스크립트에 바로 적용하실 수 있습니다.
